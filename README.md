# React Portfolio

A modern single-page portfolio built with React and Vite, styled for a bold, responsive presentation and ready for Render static deployment.

## Run locally

```bash
npm install
npm run start
```

## Deploy on Render

1. Push this repository to GitHub.
2. Create a new Render Static Site.
3. Use `npm run build` as the build command.
4. Set the publish directory to `dist`.
5. Deploy from the main branch.

## Update social links

Open `src/App.jsx` and replace the placeholder profile URLs with your actual CodeChef, LinkedIn, and GitHub accounts.
